- 토글 메뉴
  - 마이페이지 : 계정 정보를 확인할 수 있는 페이지
  - 모든프로젝트 : 게시된 모든 프로젝트를 볼 수 있는 페이지
  - 인기 프로젝트 : 일정 기간 동안 가장 후원율이 높은 프로젝트 순으로 정렬한 페이지
  - 분야별 프로젝트 : 프로젝트를 분야별로 정렬한 페이지
  - 인기 프로젝트와 분야별 프로젝트는 개발 진행중이다.
