# Klaytn-study-4

## 목표
클레이튼 덧셈게임 완성

## 활동

### + 추가 설치
```
npm install truffle-hdwallet-provider-klaytn
```

### 0. 배포
```
npm install 
truffle deploy --network testnet
```

### 1. 실행
```
npm run dev
```
